Aliasing Tools

  • Privacy (Generate credit cards for each site, lock and cancel subscriptions)
  • 1Password (Secure password manager, proprietary but audited and very reputable)
  • Bitwarden (Open source password manager, inexpensive, audited, reputable)
  • Proton Pass (Open source password manager, audited, very reputable, but less features than 1Password since Proton is newer)
  • SimpleLogin (Email alias creator mask.random@simplelogin.com)
  • Firefox Relay (Email/phone alias creator, mask@yourdomain.mozmail.com)
  • My Sudo (Email/phone/card alias creator)

Browsers

  • Mullvad (Tor fork but without using the dark web, fingerprint resistant.)
  • LibreWolf (Firefox without Mozilla analytics.)
  • Orion (From Kagi, allows Firefox AND Chrome extensions (including on iOS - but it's in beta and buggy))

Browser Extensions

  • Hyperweb (Adblock, dark mode, Picture In Picture enabler (PIP), custom styles and scripts)
  • uBlock Origin (AdBlock)
  • LibRedirect (Redirects to privacy friendly front-ends for popular sites)
  • Decentraleyes (Tracking protection from CDNs (Content Delivery Networks))
  • Privacy Possum (Better tracking protection)
  • ClearURLs (Removes extra tracking parameters from URLs)
  • Canvas Blocker (Prevents fingerprinting through JavaScript rendering an image, can mess up images sometimes if you go crazy with it)
  • OptMeowt (Sends do not sell signal to websites - Global Privacy Control (GPC) standard)
  • Kagi (Ai search engine with the ability to block sites from results, filter keywords, and have "lenses" like academic sites only)
  • NoScript (Disable JavaScript for each resource loading on a website. WILL BREAK a lot of sites that rely on JS)
  • unpaywall (Read research papers for free)
  • AdNauseam (Clicks on set number of ads in the background to screw up your advertising profile (Banned from Chrome web store because Google))
  • Web Archives (Find archived pages)
  • DuckDuckGo Privacy Essentials (It's decent, but has some holes. Great simple solution. Will try to change your search engine to duck.com lol)
  • Return YouTube Dislike (Sketchy, basically some dude scraped YouTube before they took away the feature and hosts their own database with dislike counts. IP and videos watched may be tracked by the extension retrieving data for each video)
  • Sponsorblock (Skip interaction reminders in videos and self promotion. IP and videos watched may be tracked by the extension retrieving data for each video)
  • Cookie AutoDelete (Deletes cookies after a set amount of time (Too many sweets! :D))

Browser Extensions for iOS specifically

  • Hyperweb (Adblock, dark mode, Picture In Picture enabler (PIP), custom styles and scripts)
  • xSearch (Add new default search engines, automatically redirects)
  • DeArrow (Change YouTube thumbnails to be community based and less click-bait)
  • StopTheMadness (Breaks some stuff but gives you control over sites that try to disable copy/paste, right click, drag and drop, etc.)
  • Shut Up (Removes comment sections so you don't get addicted to them)
  • Hush (Removes Popups)
  • PiPifier (Forces sites to use popup/background player)
